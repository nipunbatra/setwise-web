# 🌐 Setwise Web Interface

Professional LaTeX quiz generator with live preview, built with Streamlit.

## ✨ Features

- **📝 Split-pane editor** with questions on left, PDF previews on right
- **⚡ Live PDF generation** with LaTeX rendering  
- **🎲 Multiple quiz sets** - Generate 1-5 randomized variations
- **📋 Professional templates** - Default, compact, and minimal layouts
- **🔧 TEX downloads** - Access LaTeX source files
- **📊 Quiz metadata** - Professional headers and instructions

## 🛠️ Local Development

```bash
git clone https://github.com/nipunbatra/setwise-web.git
cd setwise-web
pip install -r requirements.txt
streamlit run streamlit_app.py
```
